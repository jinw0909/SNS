# 인스타그램2 - 모의 SNS 프로젝트
초소형 소셜네트워킹서버 + 프론트 구현

프로젝트 설명
---------
포켓몬스터를 위한 SNS서버!  
가상의 포켓몬스터가 되어 SNS로 소식을 주고받아보세요!  
원하는 사진을 올리고 마음에 드는 포스트에 댓글을 달고 좋아요를 표시할 수도 있습니다!

설계
---
* ui
* url
* db

주요기능
------
* 회원가입
* 로그인
* 글쓰기(+이미지)
* 댓글
* 좋아요
